      HMI Propriétés

      Propriétés

      sFirmware Gets or sets the firmware version.
      sInterfaceX1 Gets or sets the X1 interface address.
      sInterfaceX2 Gets or sets the X2 interface address.
      sName Gets or sets the HMI name.
      sReference Gets or sets the hardware reference.
      sVlanX1 Gets or sets the VLAN associated with X1.
      sVlanX2 Gets or sets the VLAN associated with X2.
