      MyFC Propriétés

      Propriétés

      iID Gets or sets the FC ID (number).
      iNbBloc Gets or sets the number of blocks in this FC.
      iNbLTU Gets or sets the number of LTU blocks in this FC.
      iNetwork Gets or sets the network number.
      (Hérité de MyBloc)
      oInternalBlocs Gets the list of internal blocs (FCs, FBs, etc.) used inside this FC.
      sName Gets or sets the block name.
      (Hérité de MyBloc)
      sType Gets or sets the block type (e.g., OB, FC, FB).
      (Hérité de MyBloc)
