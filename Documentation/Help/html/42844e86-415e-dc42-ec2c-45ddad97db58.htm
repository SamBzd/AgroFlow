      Propriétés

      CurrentVersion Gets or sets the current TIA Portal version. Triggers the VersionChanged event if the value changes.
      IsFirstSelection Indicates if it is the first version selection in the current session.

      Méthodes

      AreAllDllsAvailable Checks if all required Openness DLLs are available for the current TIA version.
      GetOpennessDllPath Gets the full path for a requested Openness DLL for the current TIA version.
      SetVersion Sets the current TIA Portal version.

      Événements

      VersionChanged Event triggered when the TIA version changes. The new version string is provided as event data.

      Champs

      _currentVersion Holds the current selected TIA Portal version.
      OpennessDlls List of required Openness DLLs for TIA.
