      PLC_ProjectDefinitions Propriétés

      Propriétés

      bNewNameBlockOnlyForTagNodeId Flag pour valider la génération des NodeId uniquement avec le nouveau nom dans la famille.
      bWithUITiaPortal Indique si l'interface graphique de TIA Portal doit être visible.
      sCommentarBlocParameterMarck Marque de repère des paramètres d'un bloc pour accès OPC UA.
      sCommentarTagVariableMarck Marque de repère d'un tag pour accès OPC UA.
      sDBNameMappingOPCUA Nom du DB OPC UA de mapping.
      sFamillyStrResearchNewBlocName Chaine définissant le nouveau nom de bloc.
      sFamilyBlocMarck Marque de repère des blocs OPC UA au niveau des blocs.
      sOPCUAServerNamespace Namespace pour le serveur OPC UA.
      sPathApplication Chemin complet de l'application.
      sPLCS71500HTargetStationName Nom de la station S7-1500 H cible pour l'accès en OPC UA.
      sRootFolderBlocOPCUAServer Nom du premier dossier pour la définition des blocs dans l'arborescence du serveur OPC UA.
      sRootFolderTagsOPCUAServer Nom du premier dossier pour la définition des tags dans l'arborescence du serveur OPC UA.
