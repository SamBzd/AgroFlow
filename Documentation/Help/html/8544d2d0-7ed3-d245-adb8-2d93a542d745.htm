      Remarques

      This method iterates through all device groups and devices in the project, filtering out SINAMICS devices. For each I/O module identified (matching "IM " in the type name), it extracts details such as name, reference, range, and network configuration (IP address, VLAN, and master controller). It also analyzes the I/O submodules to count the number of digital and analog inputs/outputs. Found I/O modules are added to the oInOuts collection if all required attributes are available.
