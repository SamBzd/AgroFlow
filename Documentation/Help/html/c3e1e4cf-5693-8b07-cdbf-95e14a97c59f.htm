      TIAAssemblyLoader Méthodes

      Méthodes

      ForceLoadAssemblies Forces the loading of all main Openness assemblies for the current TIA version. Logs success or error information to the console.
      ResolveAssembly Event handler for resolving TIA Openness assemblies dynamically.
      SetupControl Configures a control to use dynamic assembly resolving and handles unloading on version change.
      UnloadControl Unloads and disposes a control when the TIA version changes.
      UnloadControlImpl Helper method for removing and disposing a control.
