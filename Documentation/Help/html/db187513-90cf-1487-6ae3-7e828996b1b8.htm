      Project Represents a project containing PLCs, HMIs, SCADAs, and related project information.
      ReceptionDeProjet UserControl for the "Project Reception" function in AgroFlow.

      This module allows users to:

      • Load and analyze a TIA Portal project.
      • Verify compliance and retrieve detailed information (project info, PLCs, settings, etc.).
      • Export all relevant configuration and hardware data into a standardized Excel file for traceability and documentation.
      Typical workflow:
      1. Select a CDC Excel file (project requirements/specs).
      2. Select the TIA Portal project to analyze.
      3. Run the verification step, which collects and displays project information and device details.
      4. Export the gathered data back into the provided Excel file for archiving or transfer.
      This feature ensures project conformity to standards (Agro Mousquetaires, LTU library, etc.) and provides automatic documentation of PLC hardware and configuration.

      Remarques

      Key components:
      • ExploreTiaPLC: Handles TIA Portal project exploration and selection.
      • CompareTIA: Compares and extracts device information from the selected project.
      • PLC_ProjectDefinitions: Holds configuration and mapping definitions for the PLC project.
