      MyInOut Propriétés

      Propriétés

      iAI Gets or sets the number of analog inputs (AI).
      iAQ Gets or sets the number of analog outputs (AQ).
      iDI Gets or sets the number of digital inputs (DI).
      iDQ Gets or sets the number of digital outputs (DQ).
      sGamme Gets or sets the product range or series of the I/O module.
      sInterfaceX1 Gets or sets the X1 interface address.
      sMasterName Gets or sets the name of the master device associated with this I/O module.
      sName Gets or sets the name of the I/O module.
      sReference Gets or sets the hardware reference of the I/O module.
      sVlanX1 Gets or sets the VLAN associated with X1.
