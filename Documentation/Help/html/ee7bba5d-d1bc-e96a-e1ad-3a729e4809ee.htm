MyOB Propriétés

Le type MyOB expose les membres suivants.

Propriétés
 NomDescription
iID Gets or sets the OB ID (number).
iNbBloc Gets or sets the number of blocks in this OB.
iNbLTU Gets or sets the number of LTU blocks in this OB.
iNbNetwork Gets or sets the number of networks in this OB.
iNetwork Gets or sets the network number.
(Hérité de MyBloc)
Ltu100 Gets or sets the LTU block percentage in this OB.
oBlocList Gets the list of contained blocs (FC/FB/Instruction).
sName Gets or sets the block name.
(Hérité de MyBloc)
sType Gets or sets the block type (e.g., OB, FC, FB).
(Hérité de MyBloc)
