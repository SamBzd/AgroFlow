Méthodes
 NomDescription
ChooseTiaProject Allows the user to select a TIA Portal project, either currently open or by specifying a path.
EnumBlocksInFolderBlocks Enumerates all blocks inside a folder and exports FC blocks for OPC UA parameter extraction. Adds blocks to the internal structure and node tree.
EnumerateBlocksAndParameterOPCUAMarck Enumerates all PLC blocks and parameters with the OPC UA mark for the given station.
EnumerateBlockUserFolders Recursively enumerates all subfolders within a block user folder and processes their blocks and structures.
EnumerateFoldersBlocksParametersAndTagsForOPCUA Enumerates all folders, blocks, tags, and parameters for OPC UA export and mapping for a specific PLC station in the TIA project.
EnumerateMembers Enumerates all members (parameters, blocks, wires) found in a compile unit XML node, maps connections, and fills the data collection list for diagnostic code generation.
EnumerateTagsWithOPCUAMarck Enumerates all tags contained in tag tables with the OPC UA attribute. Scans all tag tables at the root and recursively in folders.
EnumerateTagTableUserGroup Recursively enumerates all subgroups within a tag table user group and scans contained tag tables.
EnumerateVariableSystemTags Enumerates system variables for diagnostics and adds them to the project configuration.
EnumTagTableInFolderTagTable Enumerates all tag tables in a given tag table user group and scans their variables.
